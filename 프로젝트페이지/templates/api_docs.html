국회의원 API 문서

1. 전체 의원 목록

GET /

현재(22대) 국회의원 전체를 HTML로 보여줍니다.

2. 정당별 HTML 페이지

GET /party/<party_name>

특정 정당에 소속된 국회의원 목록을 HTML로 보여줍니다.

예시: /party/더불어민주당

3. 정당별 JSON API

GET /api/party/<party_name>

특정 정당에 소속된 국회의원을 JSON 형태로 반환합니다.

예시: /api/party/국민의힘

응답 예시:

[
  {
    "name": "홍길동",
    "party": "국민의힘",
    "image_url": "https://..."
  },
  ...
]

4. 지원되는 정당 이름 목록

  • 국민의힘
  • 더불어민주당
  • 개혁신당
  • 기본소득당
  • 진보당
  • 무소속
  • 조국혁신당
  • 사회민주당

본 API는 국회공공데이터포털을 기반으로 합니다.
